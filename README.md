## Tentang

Repositori ini berisi kumpulan website jemaat GKI di Tanah Papua. Tujuannya adalah untuk memudahkan jemaat dan masyarakat umum dalam mengakses informasi terkait pelayanan GKI di berbagai wilayah Papua.
